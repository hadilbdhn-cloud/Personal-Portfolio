# Hadil Boudehane — Personal Portfolio Website

A responsive personal portfolio website built with HTML and CSS, showcasing skills, projects, and contact information.

## 📁 Project Structure

```
portfolio/
├── site.html        # Main HTML file
├── styles.css       # Stylesheet
└── images/
    ├── photo.jpeg   # Hero section background
    ├── audit.jpeg   # Project card image
    └── algeria.jpeg # Project card image
```

## ✨ Features

- **Fixed navigation header** with smooth scroll links to each section
- **Hero section** with a full-viewport background image and intro text
- **Services section** showcasing 8 skill cards (Bash, Web Dev, C, Python, ML, CS, AI, Cybersecurity)
- **Projects section** displaying 4 project cards with images and GitHub links
- **Contact section** with email and phone details
- **Footer** with copyright and social media icons
- **Responsive layout** for screens up to 1023px wide

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, media queries, transitions)
- [Google Fonts — Raleway](https://fonts.google.com/specimen/Raleway)
- [Font Awesome 7.0.1](https://fontawesome.com/) (icons)

## 🚀 Getting Started

1. Clone or download this repository.
2. Place your images in an `images/` folder next to `site.html`:
   - `photo.jpeg` — hero background
   - `audit.jpeg` — System Analysis project thumbnail
   - `algeria.jpeg` — History of Algeria project thumbnail
3. Open `site.html` in any modern browser — no build step required.
